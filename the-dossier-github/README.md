# The Dossier Indonesia

**Cultural Intelligence & Research Lab**  
Platform intelijen budaya yang mengubah riset akademis menjadi produk strategis bernilai tinggi untuk industri fashion dan lifestyle.

## Tentang

The Dossier Indonesia adalah media editorial berbasis riset yang menjembatani dunia akademis dan industri fashion & lifestyle di Indonesia. Setiap artikel lahir dari skripsi dan penelitian ilmiah yang telah divalidasi secara akademis, lalu ditransformasi menjadi wawasan yang langsung dapat digunakan oleh para pelaku industri.

**Tagline:** *Dari ruang riset, untuk industri.*

## Website

Website ini adalah single-page publication built with pure HTML, CSS, dan JavaScript — tanpa framework eksternal. Dioptimalkan untuk GitHub Pages.

### Cara Deploy ke GitHub Pages

1. Fork atau clone repository ini
2. Masuk ke **Settings → Pages**
3. Pilih branch `main` dan folder `/ (root)`
4. Klik **Save**
5. Website akan live di: `https://[username].github.io/the-dossier-indonesia/`

### Custom Domain (opsional)

Untuk menggunakan domain custom (contoh: `thedossier.id`):
1. Tambahkan file `CNAME` berisi nama domain kamu
2. Arahkan DNS domain ke GitHub Pages (lihat [dokumentasi GitHub](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site))
